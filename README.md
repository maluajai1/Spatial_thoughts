## NDBI = Normal Difference Built-up Index 

NDBI = (SWIR – NIR) / (SWIR + NIR)

For Landsat 8 data, NDBI = (Band 6 – Band 5) / (Band 6 + Band 5)

## NBR =  Normalized Burn Ratio (NBR)

NBR = (Band 5 – Band 7) / (Band 5 + Band 7).
